# Javascript starter

Starter monorepo

## Usage
Before installing any packages from this repo run
```sh
npm login --registry=https://npm.pkg.github.com
```
then add this line to your `.npmrc`
```
@damienmortini:registry=https://npm.pkg.github.com
```
